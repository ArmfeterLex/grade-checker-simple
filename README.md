# Grade Checker Simple

Простая программа на Python для проверки корректности школьной оценки.

## Функциональность

- Ввод оценки с клавиатуры
- Проверка в диапазоне от 2 до 5
- Вывод введённой оценки, если она корректна
- Сообщение "Слишком маленькая оценка" при вводе меньше 2
- Сообщение "Слишком большая оценка" при вводе больше 5

## Пример работы

Введите оценку: 4
4

Введите оценку: 1
Слишком маленькая оценка

Введите оценку: 10
Слишком большая оценка

## Куда можно встроить

- В электронный дневник
- В систему проверки ввода данных
- В учебные материалы по условным операторам
- Как пример валидации пользовательского ввода

## Требования

- Python 3.x

## Примечание

Программа предполагает ввод целого числа. При вводе текста или дробного числа возникнет ошибка — для production-версии требуется дополнительная обработка исключений.
